---

## 🛠️ My BI Architecture Blueprint
Every project in this repository strictly adheres to enterprise-level development standards:

*   Roche's Maxim: I transform data as far upstream as possible (SQL/Power Query/Excel) to keep models lightweight.
*   Star Schema Design: No flat tables. All models utilise optimised Fact and Dimension tables for maximum performance.
*   DAX Optimisation: I write clean, variables-based (`VAR`) DAX, avoiding calculated columns in favour of dynamic measures.
*   UX/UI Best Practices: Clean visual hierarchies and seamless navigation designed for busy executives.

## 🧰 Specific Skills Demonstrated

*   Power BI: Power Query ETL, DAX Measures, Visualisations, Storytelling.
*   SQL Server: Data Extraction, Data Cleansing, Joins, Aggregations, String Functions.
*   Excel: Pivot/UnPivot Analysis.
*   Data Analysis: Interpreting complex real-world data sets.
*   Data Visualisation and Reporting: Presenting complex numbers and metrics in an easy-to-understand visual format for decision makers.
